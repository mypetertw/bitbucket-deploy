# bitbucket-deploy
### Php Script for Git Pull in your web server

-

## Getting Started

Put `pull.php` to your `/` of project.

## Setting

Setting your Bitbucket information.

```php
# SETTING
define('BITBUCKET_NAME', '');
define('GIT_PASSWORD', '');
define('REPOSITORY_NAME', '');
```

## Usage

```sh
$ php pull.php
```
